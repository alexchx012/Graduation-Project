# 2026-2-24 工作日志

## 完成内容

### 1. 新增纯评估脚本（不做 PPO 更新）

- 新增文件：`scripts/go1-ros2-test/eval.py`
- 目标：对指定 checkpoint 做纯推理评估，避免 `train.py + Play` 路径继续更新策略导致统计漂移。
- 关键能力：
  - 加载指定模型并仅执行 inference step loop；
  - 复用 ROS2 bridge（订阅 `/go1/cmd_vel`）；
  - 统计 `cmd_vx`、`vx_meas`、`vx_abs_err`、`time_out/base_contact`；
  - 输出 pass/fail 结论与 JSON 汇总（`--summary_json`）。

### 2. 执行 `model_1498.pt` 的评估回放验证

- 评估模型：
  - `scripts/go1-ros2-test/run/VF-go1-Formal-Training/logs/rsl_rl/unitree_go1_flat/2026-02-23_21-44-06_vf_go1_stage1_vx06/model_1498.pt`
- 评估配置：
  - `task=Isaac-Velocity-Flat-Unitree-Go1-ROS2Cmd-Play-v0`
  - `num_envs=1`
  - `eval_steps=2000`
  - `warmup_steps=300`
  - WSL 端 ROS2 持续发布 `vx=1.0, vy=0.0, wz=0.0`（50Hz）
- 汇总文件：
  - `scripts/go1-ros2-test/run/VF-go1-Formal-Training/logs/eval_model_1498_summary.json`

### 3. 核心评估结果

- `pass = true`
- `mean_cmd_vx = 0.9853`
- `mean_vx_meas = 0.9985 m/s`
- `mean_vx_abs_err = 0.0501 m/s`
- `p95_vx_abs_err = 0.0662 m/s`
- `stable_ratio(|vx_err|<=0.1) = 0.9771`
- `mean_timeout_rate = 0.00118`
- 结论：两阶段模型在纯评估脚本下可稳定跟踪 `vx=1.0 m/s`，不需要先改超参数重训。
