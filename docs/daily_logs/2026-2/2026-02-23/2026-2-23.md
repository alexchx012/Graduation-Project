## 完成内容

### 1. 单阶段 vs 两阶段训练对比实验

基于 2026-02-20 完成的训练框架和奖励调优，今天运行了两组对比实验：

| 实验   | 配置                                                   | 总迭代数 |
| ------ | ------------------------------------------------------ | -------- |
| 单阶段 | vx=1.0 m/s, 1500 iter                                  | 1500     |
| 两阶段 | Stage-1: vx=0.6, 600 iter → Stage-2: vx=1.0, 900 iter | 1499     |

## 训练结果对比

### 核心指标（后 100 轮平均）

| 指标                            | 单阶段 (1500)  | 两阶段 (600+900)     | 优胜      |
| ------------------------------- | -------------- | -------------------- | --------- |
| **vx_meas** (实测速度)    | 1.2313 m/s     | **0.9886 m/s** | ✅ 两阶段 |
| **vx_abs_err** (跟踪误差) | 0.2932 m/s     | **0.0644 m/s** | ✅ 两阶段 |
| mean_reward                     | 2.05           | **67.68**      | ✅ 两阶段 |
| mean_episode_length             | 53.05          | **999.45**     | ✅ 两阶段 |
| cmd_timeout_count               | **2.11** | 43.83                | ✅ 单阶段 |
| cmd_zero_fallback_count         | **1.66** | 28.35                | ✅ 单阶段 |

### 详细训练曲线

**单阶段 (1500 iter)**

| 指标                | 初始值 | 最终值 |
| ------------------- | ------ | ------ |
| mean_reward         | -0.279 | 2.067  |
| mean_episode_length | 14.03  | 53.04  |
| vx_meas             | -0.097 | 1.2366 |
| vx_abs_err          | 1.0553 | 0.2674 |

**两阶段 (600+900 iter)**

| 指标                | 初始值  | 最终值  |
| ------------------- | ------- | ------- |
| mean_reward         | -0.1203 | 67.1155 |
| mean_episode_length | 14.03   | 1000.0  |
| vx_meas             | -0.099  | 0.9934  |
| vx_abs_err          | 0.674   | 0.0348  |

## 结论

### 两阶段训练显著优于单阶段

1. **速度跟踪精度**：两阶段 vx_abs_err = 0.0644 m/s，单阶段 = 0.2932 m/s，两阶段误差降低 **78%**
2. **目标达成判定**：

   - 目标：vx = 1.0 m/s
   - 两阶段实测：0.9886 m/s（误差 1.14%）→ **达标** ✅
   - 单阶段实测：1.2313 m/s（超调 23.13%）→ **未达标** ❌
3. **稳定性**：两阶段 episode_length 达到满分 1000，单阶段仅 53，说明两阶段策略更稳定
4. **奖励收敛**：两阶段 reward 67.68 vs 单阶段 2.05，差距 33 倍

### 单阶段问题分析

- 单阶段直接从零学习 vx=1.0，策略倾向于"跑得快"而非"跟得准"
- 实测速度 1.23 m/s 超调严重，说明策略没有学会精确跟踪
- episode_length 仅 53 步，频繁摔倒重置

### 两阶段优势

- Stage-1 (vx=0.6) 先建立稳定步态基础
- Stage-2 (vx=1.0) 在稳定基础上微调速度跟踪
- 最终实现 vx=0.99 m/s 的精确跟踪

### 通信指标说明

两阶段的 cmd_timeout_count 和 cmd_zero_fallback_count 较高，但这不影响最终跟踪质量。原因是两阶段训练时间更长（episode 不重置），累计的超时计数自然更多。关键指标 vx_abs_err 证明跟踪质量优秀。
