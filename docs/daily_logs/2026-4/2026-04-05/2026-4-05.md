### 2. 初次预跑失败根因确认

预跑第一次在默认沙箱下执行时，**20/20 全部失败**。

统一失败原因：

- 所有 stderr 都停在 `gym.make()` 场景创建阶段
- 错误为远程 USD 资产 `go1.usd` 拉取失败：

```text
FileNotFoundError: USD file not found at path at:
'https://omniverse-content-production.s3-us-west-2.amazonaws.com/Assets/Isaac/5.1/Isaac/IsaacLab/Robots/Unitree/Go1/go1.usd'
```

这说明：

- 失败原因**不是** `S2-S6` 场景逻辑错误
- 也**不是** baseline task / MORL task 不兼容
- 而是沙箱环境下的 Isaac 远程资产访问被阻断

后续先用单条 baseline `S2` smoke 做升级验证，确认升级权限后能成功创建环境并产出 JSON，然后在升级权限下重跑完整 `20` 次矩阵，全部通过。
